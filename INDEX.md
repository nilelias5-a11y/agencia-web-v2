# Agency Agent Index
*Last updated: 2026-05-13 by file-organizer*

## orchestration/
- **director** — Master creative and technical director. Orchestrates all specialists across 10 phases. Start every project here.
- **project-manager** — Project timeline, milestone tracking, and delivery coordination.
- **coordinator** — Cross-agent coordination and inter-phase handoffs.
- **task-router** — Classifies incoming requests and routes them to the correct specialist agent.
- **security-auditor** — Audits npm packages, GitHub repos, and external files. Issues SAFE / CAUTION / DANGER verdicts.
- **file-organizer** — Organizes agents into correct folders, detects duplicates, and keeps INDEX.md current.
- **token-saver** — Optimizes token consumption, compresses context, caches analyses, routes to cheaper models.
- **security-guardian** — Pre-activation gatekeeper for agent files. Tests against prompt injection and jailbreak patterns.

## analysis/
- **web-researcher** — Researches client's existing online presence before the briefing.
- **requirements-analyst** — Conducts client briefing interviews and produces the Project Brief.
- **ux-researcher** — User research, personas, and behavioral analysis.
- **seo-analyst** — Technical SEO audits, keyword research, and search strategy.
- **market-analyst** — Market trends, audience analysis, and sector benchmarking.
- **competitor-analyst** — Deep competitive analysis of top 5 competitors.

## design/
- **brand-designer** — Brand identity, mood boards, visual direction, and logo.
- **ux-designer** — Information architecture, user flows, and wireframes.
- **ui-designer** — Visual design, component library, and final screen designs.
- **design-system-manager** — Design tokens, component documentation, and design system maintenance.
- **prototype-designer** — Interactive prototypes and clickable wireframes.
- **mood-board-creator** — Curates visual references and style direction for client approval.

## development/
- **frontend-developer** — React/Next.js implementation, responsive layouts, and component development.
- **backend-developer** — Server-side logic, APIs, and database integration.
- **fullstack-developer** — End-to-end feature development spanning frontend and backend.
- **api-developer** — REST and GraphQL API design and implementation.
- **database-developer** — Database schema design, queries, and migrations.
- **software-recommender** — Evaluates and recommends third-party services and libraries.

## content/
- **copywriter** — Conversion-focused web copy, headlines, and CTAs.
- **seo-writer** — SEO-optimized content, blog posts, and meta descriptions.
- **content-strategist** — Content architecture, editorial calendars, and content audits.
- **social-media-manager** — Social media content, repurposing, and campaign copy.
- **translator** — Multilingual translation with next-intl and i18n support.

## performance/
- **performance-optimizer** — Core Web Vitals optimization and PageSpeed 100 targeting.
- **lighthouse-auditor** — Full Lighthouse audits across all pages.
- **core-web-vitals-specialist** — LCP, CLS, and INP deep-dive analysis and fixes.
- **caching-specialist** — Browser caching, CDN configuration, and edge caching strategy.

## deploy/
- **devops-engineer** — Deployment pipelines, environment configuration, and infrastructure.
- **ci-cd-specialist** — CI/CD pipeline setup and automation.
- **hosting-manager** — Hosting provider selection, configuration, and domain setup.
- **security-auditor** — Production security hardening and vulnerability scanning.
- **monitoring-specialist** — Uptime monitoring, error tracking, and alerting setup.

## quality/
- **quality-reviewer** — End-to-end QA across devices, browsers, and interaction states.
- **iteration-agent** — Feedback processor. Categorizes, prioritizes, and routes client revision requests.
- **comparison-engine** — A/B variant evaluator. Scores two options against criteria and delivers a verdict.

## specialists/
- **hero-specialist** — Above-the-fold experience design. Hero layouts, CTA placement, and visual hierarchy.
- **footer-architect** — Footer design, navigation architecture, legal compliance, and newsletter integration.
- **animation-premium** — GSAP and Framer Motion animations, micro-interactions, and page transitions.
- **typography-master** — Font pairing, type scale systems, variable fonts, and loading strategy.
- **color-psychologist** — Color palette strategy, brand psychology, token systems, and accessibility compliance.
- **image-curator** — Image sourcing, WebP optimization, favicon sets, and Open Graph assets.
